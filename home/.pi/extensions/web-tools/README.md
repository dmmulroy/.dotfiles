# pi-extension-web-tools

Pi extension that adds `websearch` and `webfetch` tools.

## Install

```sh
pi install ~/.dotfiles/home/.pi/extensions/web-tools
```

## Tools

- `websearch`
- `webfetch`

## Notes

- Installed from a local package path, so Pi references the source directory directly.
- Runtime dependencies are declared in `package.json` for future git/npm distribution.
