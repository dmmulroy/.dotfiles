# Footers

## Design guidelines

- Logo height between `h-5` and `h-7`
- Use `font-normal` for footer links
- Social media icons must be at least `text-gray-600` — never use `text-gray-400` or lighter

## Coding guidelines

- Always use the [placeholder content](uidotsh://ui/design-guidelines/placeholder-content) logo endpoint when no logo file is provided — never create logos from scratch with HTML or icons
